
# 🥁 DrumKit  

Um site interativo que simula uma bateria, desenvolvido para aprofundar conhecimentos em **JavaScript**, incluindo:  

- Manipulação do **DOM**  
- Uso de **callback functions**  
- Implementação de **event listeners**
